2016第一个项目

  • angular
  • javscript
  • python
  • ruby
  • 爬虫

1.重启域名

重新购买liaobaocheng.com域名，开启我的技术栈之旅。士别三日，非吴下阿蒙。

2.项目构思

我曾经思考过如何实现自由，得出的结论是时间自由->技能自由->选择自由->财务自由。
我现在处于技能自由的阶段，再持续一段时间，多做几个实战型的项目，就可以进入选择自由的阶段了，到了那个阶段我会将重心转向业务栈，其实我现在也以及在开启业务栈了，5月通过了基金考试，算是我进入金融领域的第一个倚靠，再往后，会继续通过考取证书来快速完成某个领域的知识积累。
考证本身，对自己而言是不自信的表现，对他人而言却是最简单粗暴的判断你水平的方式，学了这么就编程，我知道人得学会高内聚低耦合，不给他人添堵也就是编程思想的体现。
6月份，我会完成用ruby搭建个人网站并上线，同时开启101系列课程，第一个课程会是爬虫，从14年开始了解到爬虫这个东西，辗辗转转终于在5月份的最后几天找到了我的第五个馒头，关于爬虫的第五个馒头，前面4个馒头吃了这么久也是奇了怪了啊。

3.结语

不强为输出，感觉最近编程的状态很好。技术本身于我是第一序改变，我现在可以说是掌握了php/python/ruby/javascipt/java五门语言，接下来是实战锤炼。业务于我却是第二序改变，我开始掌握金融领域的林林总总的细节。最后，我想对自己说：简单的事情越做越困难，困难的事情越做越简单，我正走在做有门槛的事情的路上。。。
