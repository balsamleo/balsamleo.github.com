Javascript学习1220

  • 30天js

1.操作符，优先级，结合性

操作符其实也是函数，接收两个参数，返回一个结果。
操作符有优先级，优先级越高的越早调用。
结合性：从右到左，还是从左到右

2.类型强转(coercion)

因为js是动态类型的，所以这种操作很频繁。

var a = 1 + ‘2’ //结果是12(字符类型)

3.比较符

console.log(3<2<1);//true

过程是这样的：3<2结果是false，false<1结果是true，因为false也就是0.

“3” == 3 //true
“3”===3 //false
null == 0 //false
false == 0 //true

多使用===，严格相等符

4.默认值

设置默认值

6.对象和函数

对象和.操作符
对象是键值对集合，键是唯一的，值也可以是键值对。在内存中有地址，对属性有引用
对象有1.基本属性，2.对象属性，3.函数属性

7.对象和字面对象

在底层，Javascript引擎做的事情是一样的，这些写法本质上是一样的，推荐使用{}，字面常量写法。

8.伪命名空间

命名空间：变量和函数的容器（javascript没有命名空间，哈哈哈，只能faking it了）

关键是理解可以把对象当成容器，然后就有伪命名空间了。

9.JSON和字面对象

JSON：JavaScript Object Notation js对象标识
JSON和javascript中的对象不是一回事。

10.函数是对象

函数是一等公民，也是对象。其他类型可以做的事情，函数都可以做，在运行中创建函数，将函数作为参数传递等等。

函数的元素可以是1.基本类型，2.对象，3.函数，4.名字（也可以是匿名），5.表达式

11.函数声明和函数表达式

表达式：有值的代码块（无值也可以）

greet();//输出：hi

//这个是函数声明
function greet(){
    console.log('hi');
}


anonymousGreet();//输出：Uncaught TypeError，undefined is not a function

//这个是函数表达式
var anonymousGreet = function(){
    console.log('hi');
}

function log(a){
    console.log(a);
}

log(3); //输出：3

log("hello"); //输出：hello

log({
    greeting: 'hi'
}); //输出：{greeting: "hi"};

log(function(){
    console.log('hi');
});  //输出:function(){console.log('hi')};

function loog(a){
    a();
}
loog(functioni(){
    console.log('hi');
}); //输出：hi
