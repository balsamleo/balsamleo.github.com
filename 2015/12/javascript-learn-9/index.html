机器语言

计算机处理器表达的编程语言，每个编写的程序都转换成了机器语言(Machine Code)

Machine Language -> Assembly Language -> C/C++ -> Javascript

Node是用C++写的。
V8是用C++写的

ECMAScript 和 V8

标准的Javascript，有太多的Javascript引擎了。

V8源码;

Javascript Code –> C++ Program –> Machine Code

下载V8后运行(shell –print_code)可以看到javascript代码转换成机器码的过程。

服务器和客户端

Nodejs是服务器端技术，服务器提供服务，客户端请求服务，request和response，两者都是某种标准格式。CS模型。

Web服务器，浏览器，Http协议。

Javascript怎样才能做服务器语言

  • 将代码组织成可重用的代码块
  • 处理文件
  • 处理数据库
  • 网络连接
  • 接收请求和发送响应（标准格式）
  • 处理长时间工作

Module/Require/Export

模块：独立可复用的代码块，不会影响其他代码的代码块
CommonJS：一种Javascript规范
First-Class Function：其他语言类型你能做的操作，都可以对函数这么操作。
函数表达式：产生了某个结果的函数代码块
