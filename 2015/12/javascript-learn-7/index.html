Javascript学习1225

  • 30天js

搞完这个，明天开始Underscore学习，进入函数式编程的世界，更重要的是给自己一个业务场景，Underscore对我而言就是一个业务场景，先搞懂Javascript语言本身，然后再去理解AngularJS，jQuery等就简单许多了。
我曾经设想过自己要学习一门函数式语言，javascript就这么闯入了，那就好好玩吧。

Javascript Enlightenment读书笔记

1.Javascript对象

9个原生的对象构造
Number(),
String(),
Boolean(),
Object(),
Array(),
Function(),
Date(),
RegExp(),
Error()

Javascript基本上就是在这9个对象上再进行构造(还有字符串，数字，布尔值这些基本数据类型).理解这9个对象是我们高效使用Javascript这门语言威力的关键。

Math对象很奇怪，是个静态对象，而不是构造对象。直接用就行了(var x=new Math();是错误的，var x = Math.PI;是正确的).

原生对象都与全局对象关联在了一起，在浏览器中就是window

Number()/String()/Boolean()不仅仅构造对象，还能提供基本数据类型。

2.对象和属性

Object()，Array(),Function()可以包含其他对象

使用.或者[]来获取/变更对象的属性。

使用in来判断对象是是否有该属性

使用underscore.js来拓展对象的功能，lodash也可以。

7.作用域和闭包

Javascript没有块作用域。

8.函数原型属性

有四个理由，我们需要关心原型
1.原型属性用来实现原生的构造函数(Object(),Array(),Function())
2.自定义的构造函数可以添加到原型属性上，要这么做我们要先理解原型。
3.别人的实现可能是原型，你的得去修改维护
4.添加到原型链上的实现，随处可用
