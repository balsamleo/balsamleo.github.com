继续学习underscore，缺乏实战演练，体会不到这些方法的妙处啊

1.核心代码

1.闭包

整个函数是个闭包,通过传入this(也就是window对象)来改变函数作用域

2.原型赋值

Array,Object,Function都是函数，获取函数原型属性是为方便压缩。

3.数据判断

判断是否为dom，dom的nodeType属性值为1，用!!强转为布尔值(nodeType有4种)

判断是否数组，ES5有原生函数。在原生函数不存在情况下，重写一个判断函数，用call函数来改变作用域可以避免当obj没有toString函数时报错的情况。

判断是否对象，函数是对象，typeof null也是对象，所以用!!obj来区分情况

通过是否有callee来判断是否为arguments

NaN值有两个特点：1它是一个数，2不等于自己
‘+’放在变量前的作用是把后面的变量变成一个数字，把new Number()这种没有值的数字也归为NaN。

var b = new Boolean()中b也是布尔值。

void 0表示undefined
