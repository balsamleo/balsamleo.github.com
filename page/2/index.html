Javascript学习1220

1.操作符，优先级，结合性

操作符其实也是函数，接收两个参数，返回一个结果。
操作符有优先级，优先级越高的越早调用。
结合性：从右到左，还是从左到右

2.类型强转(coercion)

因为js是动态类型的，所以这种操作很频繁。

var a = 1 + ‘2’ //结果是12(字符类型)

3.比较符

console.log(3<2<1);//true

过程是这样的：3<2结果是false，false<1结果是true，因为false也就是0.

“3” == 3 //true
“3”===3 //false
null == 0 //false
false == 0 //true

多使用===，严格相等符

Javascript学习1219

1.键值对/对象

键值对：键可能多次定义，但在特定context中只有唯一的一个值。值本身可能也是键值对。
对象：键值对的集合

Address是一个对象

2.全局作用域，全局对象

js代码一旦执行，就产生了‘全局对象’和‘this’

在浏览器中 this===window===frames

3.执行上下文，变量提升

变量提升的原因:

Creation Phase(创建阶段)：
‘this’,外部环境，为变量和函数申请内存

在代码执行前，变量和函数在内存中已经存在了，代码的执行是一行一行的，a已经在内存中存在了，但是解释器不知道变量a的具体内容，所以显示’undefined’，如果是a不定义，则显示’Uncaught ReferenceError’(因为a在内存中不存在)

Javascript中，初始化值都是undefined,undefined是一种特殊值

Execution Phase(执行阶段):
链接-编译-解释-执行
